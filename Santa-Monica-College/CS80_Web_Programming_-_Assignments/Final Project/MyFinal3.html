CS80 Website Project

This page is a demo that was built with HTML/CSS and JavaScript and can also be viewed on a phone, ipad or other table, as well as destop. You may reload the page with button at the top right!

Buttons will cycle through projects. At end it will remove all remaining elements from the page. Use reload button to navigate back over previous elements.

Note:This demo of the flexbox container allows pointing clicking, dragging, and hover effects. Use button at page top to enlarge image.
